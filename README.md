### jdate library in lua

* Year
* Month
* Day
* ...

### Require
```
local jdate = require 'jdate'
```

### parameters 
```
#Y  --> سال
   
#M  --> ماه

#D  --> روز
  
#h  --> ساعت

#m  --> دقیقه
  
#s  --> ثانیه

#x  --> نام روز
  
#X  --> نام ماه
  
#F  --> نام فصل

#y  --> حیوان سال 

#z  --> ذکر روز

```
